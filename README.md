## suod 的設定
[參考資料](http://niucp03.blogspot.com/2012/02/debian-user-sudo.html)
修改檔案

```
vim /etc/sudoers
```

加入

```
user ALL=(ALL) ALL
```
設定termianl快節健，在設定快捷健那邊。
```
    Name: gnome-terminal
    Command: gnome-terminal
    Ctrl+Alt+T
```
